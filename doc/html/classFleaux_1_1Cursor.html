Fleaux::Cursor::Cursor ( Editor *  ed)

Constructor creates a new Cursor instance that belongs to the editor pointed to by ed. The optional callerIsEditor parameter is used to determine if this is being called from an Editor's constructor, or by another entity. void Fleaux::Cursor::remove ( int  length)
virtual

Removes n characters, where n is |length|. The sign of length determines the direction from the cursor in which characters are removed: (-) removes backward starting at the character behind the cursor, (+) removes forward starting at the character under the cursor.
